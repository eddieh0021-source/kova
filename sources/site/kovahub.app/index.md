FAQ

## Common questions.

Everything you need to know before getting started.

Is Kova Hub really free?+

Yes — 100%, always. Kova Hub is built to give ER:LC communities a polished CAD without any monthly costs or premium tiers.

How does the live map work?+

Paste your ER:LC PRC API key in Manage Server → Integrations. Kova Hub polls every second to pull live player positions, vehicles, and plates onto the dispatch map.

What is the 911 Event Webhook?+

PRC's Event Webhook pushes 911 calls directly to Kova Hub the moment a civilian dials in-game — it appears in dispatch with a chime automatically, no polling needed.

Does it work with Discord?+

Yes. Kova Hub uses Discord OAuth for login and supports a full audit log — every change posts to a Discord channel of your choice, with per-event filtering.

What departments are supported?+

Police, Fire/EMS, and DOT each get dedicated CAD panels. Create custom departments and divisions, assign roles, and control access per department.
